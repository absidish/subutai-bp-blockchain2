Wordpress Standalone

Or: How to set up a self-hosted Wordpress site in one minute.

The Wordpress Standalone blueprint was my first venture into creating blueprints on Subutai.

It creates a Subutai environment with one container and install MariaDB and the latest version of Wordpress into that environment. It will also setup and configure Wordpress as specified when first running the blueprint.

The source of the blueprint is available on Github https://github.com/lbthomsen/subutai-blueprint-wp

Preparation

In order to install and run your first Subutai Wordpress site, go through the following steps.

Login to Subutai Bazaar

First, go to the Subutai Bazaar https://bazaar.subutai.io

Provide your login details, or sign up if you are not already a registered user.

Register a domain

Click on your username in the top right hand corner and then click "Domains" in the pop-up menu. On the resulting page, click the green "Add new" button.

Add peers

From the Peers item in the main menu, add one or more peers to your favorites:

You are now ready to set up your first Wordpress site on Subutai.

Build your site

Now that all the prerequisites are in place, it is time to build your new Wordpress site. Go to "Products" in the right hand side menu:

Select "Wordpress Standalone" and on the product page, click on the blue "Build" button at the top.

You will now be presented with a list of blueprint variables that will need to be filled in.

Most of these variables are self explanetory and Wordpress related. When the variables are filled in press the blue Start button.

On the exposed ports page nothing needs to be changed, so simply click "Next", which will bring you to a page where you can select which peer to use.

The list of peers will contain a combination of your own peers and peers which you selected as favorites earlier. Select one (and only one) peer from the list and click "Next".

The Blueprint is now ready to run so click the Finish button to begin.

While the blueprint is building the environment, progress can be displayed by clicking the "Logs" link:

After a short while (a minute or two) the environment build should be done and the environment turns "Healty".

On the Container Port Mapping tab simply click on the domain link that you selected earlier and you will see your spankling new Wordpress site.

And there you have it - you have created a self hosted Wordpress site in a matter of minutes.
